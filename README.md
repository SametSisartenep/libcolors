# libcolors

A basic colors library for the C language.

# API

```c
void color ( const char *string ); // Print a <color> string
```

Currently supported colors:

- red
- green
- yellow
- blue
- magenta
- cyan

### Coming soon
```c
void r_print ( const char *fmt, ... ); // Print a multicolor array.
```
